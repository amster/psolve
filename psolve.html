What It Is

This is a puzzle-solver helper. Typically during puzzle hunts you'll need to translate between letters, numbers, and indexes. This is like a super code sheet.

  • Type values into the left side and it will try to solve out answers to the right.
  • Separate elements in the Input by spaces or commas.
  • Press Enter to go down 1 field, Tab to go to the next field in order
  • Some special Inputs:
    • Dots and dashes for Morse Code (--.- to get "Q", -.-. to get "C")
    • Direction pairs for semaphores (nw/sw to get "I", ne/s to get "E")
    • Braille as a string of "x" and "." (x.x.x. to get "L", .xxx.x to get "W")
